# 121. Best Time to Buy and Sell Stock

## 🧠 Approach

- Used **greedy approach** to maximize profit

- Maintain:
  - `bestBuy` → minimum price seen so far  
  - `maxProfit` → maximum profit found  

- Traverse the array:
  - If current price is greater than `bestBuy`:
    - Calculate profit:
      ```
      prices[i] - bestBuy
      ```
    - Update `maxProfit`
  - Update `bestBuy` with minimum value so far

- Return `maxProfit`


## 💡 Key Insight

- We want to **buy at lowest price** and **sell at highest price after that**

- Instead of checking all pairs:
  - Keep track of minimum price so far
  - Calculate profit at each step


## 🧩 Pattern

This problem follows **Greedy + Single Pass Optimization** pattern.

### 🔹 Used when:
- Need to maximize/minimize value
- Future decisions depend on past best values
- Problems involving profit, cost, or difference


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
int maxProfit(vector<int>& prices) {
    int maxProfit = 0;
    int bestBuy = prices[0];

    for (int i = 0; i < prices.size(); ++i) {
        if (prices[i] > bestBuy) {
            maxProfit = max(maxProfit, prices[i] - bestBuy);
        }

        bestBuy = min(bestBuy, prices[i]);
    }

    return maxProfit;
}
