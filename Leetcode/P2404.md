# 2404. Most Frequent Even Element

## 🧠 Approach

- Used **hash map(unordered map)** to store frequency of only even elements

- Traverse the array:
  - If element is even → increase its frequency in map

- Maintain:
  - `m` → maximum frequency found so far  
  - `a` → answer (most frequent even element)

- Traverse the map:
  - If frequency of current element is greater than `m`:
    - Update `m` and set `a` to current element
  - If frequency is equal to `m`:
    - Choose the **smaller element**

- If no even element exists → return `-1`


## 💡 Key Insight

- Only even elements matter → filter early  
- Use frequency map to track occurrences  
- Apply tie-breaking by choosing **minimum value**


## 🧩 Pattern

This problem follows **frequency counting + tie-breaking** pattern.

### 🔹 Used when:
- Need to find most frequent element
- Additional constraints like:
  - only specific elements (even numbers here)
  - tie-breaking rules (smallest value)


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(n)`


## 💻 Code

```cpp
int mostFrequentEven(vector<int>& nums) {
    unordered_map<int, int> map;

    for (int i : nums) {
        if (i % 2 == 0) {
            map[i]++;
        }
    }

    int a = INT_MAX;
    int m = 0;

    for (auto &pair : map) {
        if (pair.second > m) {
            m = pair.second;
            a = pair.first;
        }
        else if (pair.second == m) {
            if (pair.first < a) {
                a = pair.first;
            }
        }
    }

    if (a < INT_MAX) {
        return a;
    }

    return -1;
}
