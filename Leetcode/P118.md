# 118. Pascal's Triangle

## 🧠 Approach

- Used **nested loops** to build Pascal’s Triangle row by row

- Maintain a 2D vector `vec` to store all rows

- For each row `i`:
  - Create a new row `arr`
  - Traverse from `j = 0` to `i`

- For each element:
  - If it is the **first or last element** (`j == 0` or `j == i`) → push `1`
  - Else:
    - Value = sum of two elements from previous row  
      ```
      vec[i-1][j-1] + vec[i-1][j]
      ```

- Push the constructed row into `vec`

- Return final 2D vector


## 💡 Key Insight

- Each element (except edges) is the **sum of two elements above it**
- First and last elements of every row are always `1`


## 🧩 Pattern

This problem follows **dynamic construction / bottom-up building** pattern.

### 🔹 Used when:
- Current state depends on **previous row/state**
- Building structure step by step


## ⏱ Complexity

- **Time Complexity:** `O(n^2)`  
- **Space Complexity:** `O(n^2)`


## 💻 Code

```cpp
vector<vector<int>> generate(int numRows) {
    vector<vector<int>> vec;

    for (int i = 0; i < numRows; ++i) {
        vector<int> arr;

        for (int j = 0; j <= i; ++j) {
            if (j == 0 || j == i) {
                arr.push_back(1);
            } else {
                arr.push_back(vec[i-1][j-1] + vec[i-1][j]);
            }
        }

        vec.push_back(arr);
    }

    return vec;
}
