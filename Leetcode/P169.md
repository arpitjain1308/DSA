# 169. Majority Element

## 🧠 Approach

- Maintain two variables:
  - `a` → current **candidate**
  - `c` → **count (votes)** for the candidate

- Traverse the array:
  - If `c == 0`, select the current element as the new candidate
  - If the current element is the same as the candidate → increment `c`
  - Otherwise → decrement `c`

- This works because:
  - The majority element appears **more than n/2 times**
  - All other elements combined cannot fully cancel it out
  - Hence, the final candidate remains the majority element


## 💡 Key Insight

This uses the **Boyer–Moore Voting Algorithm**.

It works on the idea of **cancelling out different elements**, ensuring that the majority element survives.


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
int majorityElement(vector<int>& nums) {
    int a = 0;
    int c = 0;

    for (int i = 0; i < nums.size(); ++i) {
        if (c == 0) {
            a = nums[i];
        }

        if (nums[i] != a) {
            c--;
        } else {
            c++;
        }
    }

    return a;
}
