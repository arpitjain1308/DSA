# 1552. Magnetic Force Between Two Balls

## 🧠 Approach

- Used **Binary Search on Answer**

- Goal:
  - Maximize the **minimum distance** between any two balls

- Given:
  - `position[]` → positions of baskets  
  - `m` → number of balls  

- First, sort the positions

---

## 🔹 Key Idea

- We don’t search for position  
- We search for **maximum minimum distance**

- Range:
start = 1
end = max(position) - min(position)

---

## 🔹 Validity Check (isValid)

- Check if we can place `m` balls such that:
- Minimum distance between any two balls ≥ `mid`

- Strategy:
- Place first ball at first position
- Greedily place next balls at the earliest position satisfying distance

- If we can place all `m` balls → valid  
- Else → not valid  

---

## 🔹 Binary Search

- Pick `mid` (candidate distance)

- If valid:
- Store answer
- Try larger distance → `start = mid + 1`

- Else:
- Reduce distance → `end = mid - 1`

---

## 💡 Key Insight

- We try to **push balls as far apart as possible**
- Greedy placement works because:
- placing earlier leaves more space for future placements

---

## 🧩 Pattern

This problem follows **Binary Search on Answer (Maximize Minimum)** pattern.

### 🔹 Used when:
- Need to maximize minimum distance/value  
- Greedy check can validate feasibility  
- Monotonic behavior exists  

---

## ⏱ Complexity

- **Time Complexity:** `O(n log(range))`  
- **Space Complexity:** `O(1)`

---

## 💻 Code

```cpp
bool isValid(vector<int>& position, int n, int m, int mid) {
  int mag = 1;
  int lastbucket = position[0];

  for (int i = 1; i < n; ++i) {
      if (position[i] - lastbucket >= mid) {
          mag++;
          lastbucket = position[i];
      }
      if (mag == m) return true;
  }

  return false;
}

int maxDistance(vector<int>& position, int m) {
  sort(position.begin(), position.end());

  int n = position.size();
  int start = 1;
  int end = position[n-1] - position[0];
  int ans = 0;

  while (start <= end) {
      int mid = start + (end - start) / 2;

      if (isValid(position, n, m, mid)) {
          ans = mid;
          start = mid + 1;
      } else {
          end = mid - 1;
      }
  }

  return ans;
}
