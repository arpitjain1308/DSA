# 229. Majority Element II

## 🧠 Approach

- Used **Boyer-Moore Voting Algorithm (extended)**

- Key observation:
  - Elements appearing more than `n/3` times can be **at most 2**

- Maintain:
  - `a`, `b` → two candidates  
  - `c1`, `c2` → their respective counts  


### 🔹 Phase 1: Candidate Selection

- Traverse the array:
  - If current element matches `a` → increment `c1`
  - Else if it matches `b` → increment `c2`
  - Else if `c1 == 0` → assign current element to `a`
  - Else if `c2 == 0` → assign current element to `b`
  - Else → decrement both `c1` and `c2`

- This step cancels out non-majority elements


### 🔹 Phase 2: Verification

- Count occurrences of both candidates again
- Add them to result only if: count > n/3



## 💡 Key Insight

- At most **2 elements** can appear more than `n/3` times  
- All other elements get cancelled out during traversal  
- Final candidates are **potential answers**, so verification is required  


## 🧩 Pattern

This problem follows the **generalized Boyer-Moore Voting pattern**.

### 🔹 Rule:
> If an element appears more than `n/k` times, there can be at most `k - 1` such elements.

### 🔹 Application:
- For `n/2` → at most **1** candidate  
- For `n/3` → at most **2** candidates  

### 🔹 Strategy:
1. Maintain `k - 1` candidates with counts  
2. Cancel out elements that don’t match  
3. Verify candidates in a second pass  


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
vector<int> majorityElement(vector<int>& nums) {
  vector<int> vec;

  int a = 0, b = 1;
  int c1 = 0, c2 = 0;

  for (int i = 0; i < nums.size(); ++i) {
      if (nums[i] == a) {
          c1++;
      }
      else if (nums[i] == b) {
          c2++;
      }
      else if (c1 == 0) {
          a = nums[i];
          c1++;
      }
      else if (c2 == 0) {
          b = nums[i];
          c2++;
      }
      else {
          c1--;
          c2--;
      }
  }

  int ca = 0, cb = 0;

  for (int i : nums) {
      if (i == a) ca++;
      if (i == b) cb++;
  }

  if (ca > nums.size() / 3) {
      vec.push_back(a);
  }
  if (cb > nums.size() / 3) {
      vec.push_back(b);
  }

  return vec;
}
