# 11. Container With Most Water

## 🧠 Approach

- Used **two pointer approach** to maximize area

- Maintain:
  - `start` → left pointer  
  - `end` → right pointer  
  - `ans` → maximum area found  

- Traverse while `start < end`:
  - Calculate area:
    ```
    min(height[start], height[end]) * (end - start)
    ```
  - Update `ans` with maximum value

- Move pointer:
  - If left height is smaller → move `start++`
  - Else → move `end--`

- Return `ans`


## 💡 Key Insight

- Area depends on:
  - height → smaller of two bars  
  - width → distance between them  

- To increase area:
  - Move the pointer with **smaller height**
  - Because larger height might give better result ahead  


## 🧩 Pattern

This problem follows **Two Pointer Optimization** pattern.

### 🔹 Used when:
- Array is processed from both ends  
- Need to optimize brute force (reduce O(n²) to O(n))  
- Problems involving pairs and maximum/minimum values  


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
int maxArea(vector<int>& height) {
    int start = 0;
    int end = height.size() - 1;
    int ans = 0;

    while (start < end) {
        int area = min(height[start], height[end]) * (end - start);
        ans = max(ans, area);

        if (height[start] < height[end]) {
            start++;
        } else {
            end--;
        }
    }

    return ans;
}
