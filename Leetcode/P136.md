# 136. Single Number

## 🧠 Approach

- Used **bit manipulation (XOR)** to find the single element

- Maintain a variable `a`:
  - Initialize with first element

- Traverse the array:
  - Perform XOR of current element with `a`
    ```
    a ^= nums[i]
    ```

- Return final value of `a`


## 💡 Key Insight

- XOR has important properties:
  - `x ^ x = 0` (same elements cancel out)
  - `x ^ 0 = x`

- Since every element appears twice except one:
  - All duplicates cancel out
  - Only the unique element remains


## 🧩 Pattern

This problem follows **bit manipulation / XOR cancellation** pattern.

### 🔹 Used when:
- Every element appears twice except one
- Need optimal solution without extra space


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
int singleNumber(vector<int>& nums) {
    int a = nums[0];

    for (int i = 1; i < nums.size(); ++i) {
        a ^= nums[i];
    }

    return a;
}
