# 231. Power of Two

## 🧠 Approach

- Used **bit manipulation** to check if number is power of 2

- First check:
  - If `n <= 0` → return `false`

- Then apply condition: n & (n - 1)

- If result is `0` → number is power of 2  
- Else → not a power of 2  


## 💡 Key Insight

- Power of 2 numbers have **only one set bit (1)** in binary

Examples: 
1 -> 0001
2 -> 0010
4 -> 0100
8 -> 1000

- When we do: n & (n - 1)
→ it removes the last set bit

- So:
- If only one set bit → result becomes `0`
- Otherwise → not `0`


## 🧩 Pattern

This problem follows **bit manipulation (set bit check)** pattern.

### 🔹 Used when:
- Checking powers of 2  
- Counting or removing set bits  
- Binary-based problems  


## ⏱ Complexity

- **Time Complexity:** `O(1)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
bool isPowerOfTwo(int n) {
  return n > 0 && (n & (n - 1)) == 0;
}
