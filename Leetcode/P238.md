# 238. Product of Array Except Self

## 🧠 Approach

- Used **prefix + suffix product** to avoid division

- Create result array `ans` initialized with `1`

### 🔹 Prefix pass (left → right)
- For each index `i`:
ans[i] = product of all elements before i
- Build it using:
ans[i] = ans[i-1] * nums[i-1]

### 🔹 Suffix pass (right → left)
- Maintain `suffix` product
- For each index `i`:
- Multiply current value with suffix:
  ```
  ans[i] *= suffix
  ```
- Update suffix:
  ```
  suffix *= nums[i]
  ```

- Return `ans`


## 💡 Key Insight

- For each index:
answer[i] = (product of left elements) * (product of right elements)
- Instead of storing both arrays:
- Store prefix in `ans`
- Use a single variable for suffix → saves space


## 🧩 Pattern

This problem follows **Prefix-Suffix / Precomputation** pattern.

### 🔹 Used when:
- Need product/sum excluding current index  
- Division is not allowed  
- Optimize brute force O(n²) → O(n)


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)` (excluding output array)


## 💻 Code

```cpp
vector<int> productExceptSelf(vector<int>& nums) {
  vector<int> ans(nums.size(), 1);

  // prefix
  for (int i = 1; i < nums.size(); ++i) {
      ans[i] = ans[i-1] * nums[i-1];
  }

  // suffix
  int suffix = 1;
  for (int i = nums.size() - 2; i >= 0; i--) {
      suffix *= nums[i+1];
      ans[i] *= suffix;
  }

  return ans;
}
