# 50. Pow(x, n)

## 🧠 Approach

- Used **Binary Exponentiation (Fast Power)** to compute power efficiently

- Convert exponent `n` to long (`binform`) to handle edge cases (like `INT_MIN`)

- If exponent is negative:
  - Convert it to positive
  - Replace `x` with its reciprocal (`1/x`)

- Maintain:
  - `ans` → final result  

- Traverse while `binform > 0`:
  - If current power is odd:
    ```
    ans *= x
    ```
  - Square the base:
    ```
    x *= x
    ```
  - Reduce power:
    ```
    binform /= 2
    ```

- Return final `ans`


## 💡 Key Insight

- Instead of multiplying `x` n times → use **divide and conquer**
- Break power into binary form:
  - Even power → square the base  
  - Odd power → multiply once and reduce  


## 🧩 Pattern

This problem follows **Binary Exponentiation / Divide and Conquer** pattern.

### 🔹 Used when:
- Large exponent calculations  
- Power computations  
- Modular exponentiation  


## ⏱ Complexity

- **Time Complexity:** `O(log n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
double myPow(double x, int n) {
    long binform = n;

    if (binform < 0) {
        binform = -binform;
        x = 1 / x;
    }

    double ans = 1;

    while (binform > 0) {
        if (binform % 2 == 1) {
            ans *= x;
        }
        x *= x;
        binform /= 2;
    }

    return ans;
}
