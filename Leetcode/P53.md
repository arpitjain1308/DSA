# 53. Maximum Subarray

## 🧠 Approach

- Used **Kadane's Algorithm** to find maximum subarray sum in linear time

- Maintain:
  - `currsum` → current running sum  
  - `maxsum` → maximum sum found so far  

- Traverse the array:
  - Add current element to `currsum`
  - Update `maxsum` with maximum of `currsum` and previous `maxsum`
  - If `currsum` becomes negative → reset it to `0`


## 💡 Key Insight

- We do not include negative contributions in sum  
- If current sum becomes negative, it will reduce future sums  
- So, reset and start fresh from next element  


## 🧩 Pattern

This problem follows **Kadane’s Algorithm (Dynamic Programming)** pattern.

### 🔹 Used when:
- Finding maximum/minimum subarray sum  
- Continuous subarray problems  
- Optimization over linear traversal  


## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
int maxSubArray(vector<int>& nums) {
    int currsum = 0;
    int maxsum = INT_MIN;

    for (int i = 0; i < nums.size(); ++i) {
        currsum += nums[i];
        maxsum = max(maxsum, currsum);

        if (currsum < 0) {
            currsum = 0;
        }
    }

    return maxsum;
}
