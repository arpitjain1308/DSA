# 540. Single Element in a Sorted Array

## 🧠 Approach

- Used **binary search with index parity (even/odd)**

- Given:
  - Array is sorted  
  - Every element appears twice except one  

- Maintain:
  - `start` → 0  
  - `end` → n - 1  

- Traverse while `start <= end`:
  - Find `mid`


### 🔹 Key Observation

- Before the single element:
  - pairs start at **even index**
    ```
    (0,1), (2,3), (4,5) ...
    ```

- After the single element:
  - pattern breaks → pairs start at **odd index**


### 🔹 Logic

- If `nums[mid]` is not equal to neighbors → return it

- If `mid` is even:
  - If `nums[mid] == nums[mid-1]` → move left  
  - Else → move right  

- If `mid` is odd:
  - If `nums[mid] == nums[mid+1]` → move left  
  - Else → move right  


## 💡 Key Insight

- The single element **breaks the pairing pattern**
- Use index parity to decide search direction


## 🧩 Pattern

This problem follows **Binary Search with Pattern Break / Parity Check**

### 🔹 Used when:
- Sorted array with duplicates  
- One unique element  
- Pattern shift occurs at answer  


## ⏱ Complexity

- **Time Complexity:** `O(log n)`  
- **Space Complexity:** `O(1)`


## 💻 Code

```cpp
int singleNonDuplicate(vector<int>& nums) {
    int start = 0;
    int end = nums.size() - 1;

    if (end == 0) return nums[0];

    while (start <= end) {
        int mid = start + (end - start) / 2;

        if (mid == 0 && nums[0] != nums[1]) return nums[0];
        if (mid == nums.size() - 1 && nums[nums.size()-2] != nums[nums.size()-1]) return nums[nums.size()-1];

        if (nums[mid-1] != nums[mid] && nums[mid] != nums[mid+1]) return nums[mid];

        else if (mid % 2 == 0) {
            if (nums[mid-1] == nums[mid]) {
                end = mid - 1;
            } else {
                start = mid + 1;
            }
        }
        else {
            if (nums[mid+1] == nums[mid]) {
                end = mid - 1;
            } else {
                start = mid + 1;
            }
        }
    }

    return -1;
}
