# 26. Remove Duplicates from Sorted Array

## 🧠 Approach

- Used **two pointers**:
  - `a` → tracks the index of the **last unique element**
  - `b` → acts as a **scanning pointer** to check each element

- Traverse the array using pointer `b` until it reaches the end.

- Maintain a variable `unique_count`:
  - Tracks the number of unique elements
  - Also represents the **next position to place a new unique element**

- For each step:
  - If `nums[a] == nums[b]`:
    - Move `b` forward (duplicate found)
  - Else:
    - A new unique element is found
    - Place it at the next position (`unique_count`)
    - Increment `a`, `b`, and `unique_count`

- Return `unique_count` as the final answer.

## ⏱ Complexity

- **Time Complexity:** `O(n)`  
- **Space Complexity:** `O(1)` (in-place)

## 💻 Code[CPP]

int removeDuplicates(vector<int>& nums) {
    int a = 0;
    int b = 1;
    int unique_count = 1;

    while (b < nums.size()) {
        if (nums[a] == nums[b]) {
            b++;
        } else {
            nums[unique_count] = nums[b];
            unique_count++;
            b++;
            a++;
        }
    }

    return unique_count;
}
